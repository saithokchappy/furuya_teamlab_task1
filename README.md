# チームラボオンラインスキルアップSTEP1最終課題_古谷


Todoリスト

## Description

フォームに文章を入れて追加ボタンを押すとリストが作成される
簡素なTodoリストです。

## Features

- 削除ボタンを押すと全リストが削除され、ねぎらいの言葉が通知されます。

## Requirement
- Google Chrome
## Usage

1. 任意のローカルフォルダで全てのファイルを配置する。
2. todolist_furuya.htmlを開く。
3. フォームに文章を入れ追加ボタンを押すとリストが更新される。
4. 削除ボタンを押すとリストが全削除される。
